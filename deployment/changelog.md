## 1.2.0

### Added

- You can now set an activity when scheduling — choose what you plan to do (Quest, PVP, Normal Dungeon, Heroic Dungeon, Raid, Farm, or Chill).
- For dungeons, raids, and PVP, a second dropdown lets you pick the specific instance or battleground.
- The player list now shows each player's activity in a new column, with the specific instance shown below the activity type when set.
- The activity column is sortable — click the header to sort by what players are doing.
- Two new filter dropdowns let you filter the player list by activity type and, when applicable, by specific instance.
