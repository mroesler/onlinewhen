  <action>
Replace the placeholder onChange handler in the primaryActivityFilterBtn construction (from Plan 05-02)
with the full cascade logic per D-06. The onChange receives `val` which is either nil (Any Activity)
or a label string like "Normal Dungeon".

Replace the onChange function body with:

```lua
    function(val)
        filterPrimaryActivity = val
        filterExactActivity = nil
        currentPage = 1
        if exactActivityFilterBtn then
            if val == nil then
                -- "Any Activity" selected: reset exact to disabled
                exactActivityFilterBtn.setChoices({ { label = "Any Exact Activity", value = nil } }, "Any Exact Activity")
                exactActivityFilterBtn.setActive(false)
            else
                local subs = OW.ACTIVITY_SUBS[val]
                if subs and #subs > 0 then
                    -- Activity has sub-types: enable exact dropdown with sub-type choices
                    local exactChoices = { { label = "Any Exact Activity", value = nil } }
                    for _, sub in ipairs(subs) do
                        exactChoices[#exactChoices + 1] = { label = sub, value = sub }
                    end
                    exactActivityFilterBtn.setChoices(exactChoices, "Any Exact Activity")
                    exactActivityFilterBtn.setActive(true)
                else
                    -- Quest/Farm/Chill: no sub-types, keep exact disabled (per FILT-06)
                    exactActivityFilterBtn.setChoices({ { label = "Any Exact Activity", value = nil } }, "Any Exact Activity")
                    exactActivityFilterBtn.setActive(false)
                end
            end
        end
        TL.Refresh()
    end
```

Key behaviors:
- `filterExactActivity = nil` always cleared when primary changes (per FILT-05 / D-06)
- `OW.ACTIVITY_SUBS[val]` lookup determines if sub-types exist
- `#subs > 0` check: Quest/Farm/Chill have empty tables, so this correctly disables exact
- Sub-type values are plain strings (`value = sub`), matching entry.exactActivity format
- "Any Exact Activity" is the placeholder label in all states per D-02
