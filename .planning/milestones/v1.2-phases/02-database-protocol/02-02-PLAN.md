Make two changes in Network/Protocol.lua:

**Change 1 — Fix split() (D-01):**
Replace the split function (currently around lines 148-154) from:
```lua
local function split(str, sep)
    local parts = {}
    for part in str:gmatch("([^" .. sep .. "]+)") do
        table.insert(parts, part)
    end
    return parts
end
```
to:
```lua
local function split(str, sep)
    local parts = {}
    for part in (str .. sep):gmatch("([^" .. sep .. "]*)" .. sep) do
        table.insert(parts, part)
    end
    return parts
end
```

The key difference: `([^;]*)` (zero or more) instead of `([^;]+)` (one or more), and prepending `(str .. sep)` as a sentinel. This preserves empty tokens so `"a;;b"` yields `{"a", "", "b"}` instead of `{"a", "b"}`.

**Change 2 — Extend SerializeANN (D-02):**

This makes the wire format 12 fields: `1;ANN;Name;Realm;Spec;Level;OnlineAtUTC;TzId;UpdatedUTC;Class;primaryActivity;exactActivity`
